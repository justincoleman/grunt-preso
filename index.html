What's Grunt?

Grunt is a JavaScript task-runner.

What?

Basically, it's a tool that automates your workflow so that you can focus on the important stuff.

Really? All of the important stuff? Yeah right.

No really. Pretty much anything in your workflow can be automated by Grunt.

  • Sass compilation / CSS minification / Autoprefixer
  • JSHint / JS Uglify
  • Image optimization
  • File concatenation
  • And so much more!
